前言

香港城市大学教授关于天线相关知识的讲座主要介绍了当前天线领域比较前沿的一些研究，与现在实验室的项目也有一定的耦合度，其在相关领域取得的成果也是非常值得我们借鉴和学习的。 # 超材料天线 超材料（metamaterial）指的是一些具有人工设计的结构并呈现出天然材料不能的超常物理性质的复合材料，一般包括左手材料、光子晶体、超磁性材料、金属水等。 ## 周期性平行板传输线 周期性平行板天线是超材料天线的一种,这里介绍此类型的传输线主要是通过其原理分析简单阐述超材料天线的工作原理。假设该平行板的周期长度为d，缺口宽度为g，金属板宽度为w，上下两层平行板之间通过介质基板隔开，介质基板厚度为h，宽度同样为w。则该周期性平行板传输线的等效阻抗如图所示，其中\(Z_{a}\)为传输线特征阻抗，\(β_{a}\)为传输线传播常数。当传输线产生LC谐振时，电磁波不能进行传播，从而实现负磁导率。
周期性平行板传输线
周期性平行板传输线等效阻抗

根据传输线理论可知，该周期性平行板传输线是一个TEM波传输线，其所对应的特征阻抗Z_a的计算公式为： \[Z^{a}= \frac{\eta h}{w}= 377\frac{h}{w\sqrt{\varepsilon _{r}}}\] 传输线的单位分布电感和电容可表示为： \[ \left\{\begin{matrix} L^{'}_{R}=\mu \frac{h}{w}\\ C^{'}_{R}=\varepsilon \frac{w}{h} \end{matrix}\right. \] 在终端短路或者开路的边界条件下，当传输线发生谐振时应满足： \[\beta _{a}L=\beta _{a}\left ( Nd \right )=m\pi \left ( m=0,1,2,\cdots \right )\] 根据周期性传输线理论，其色散关系式为： \[\cos \beta _{a}d=1-\frac{1}{2}\left ( \frac{\omega ^{2}-\omega _{MZR}^{2}}{\frac{1}{L_{R}C_{R}}} \right )\] 其中 \[\omega _{MZR}=\frac{1}{\sqrt{L_{R}C_{R}}} L_{R}=L_{R}^{'}d,C_{R}=C_{R}^{'}d\] 由于当传输线终端开路时，其输入阻抗表示为： \[ Z_{in}^{open}=-jZ_{c}\cot \left ( \beta _{a}L \right )\approx -jZ_{c}\frac{1}{\beta _{a}L}\left ( \beta _{a}\rightarrow 0 \right )=-j\sqrt{\frac{Z^{'}}{Y^{'}}}\left ( \frac{1}{-j\sqrt{Z^{'}Y^{'}}} \right )\frac{1}{L}=\frac{1}{NY} \] 当传输线终端短路时，其输入阻抗可表示为： \[ Z_{in}^{short}=-jZ_{c}\tan \left ( \beta _{a}L \right )\approx -jZ_{c}\beta _{a}L\left ( \beta _{a}\rightarrow 0 \right )=-j\sqrt{\frac{Z^{'}}{Y^{'}}}\left ( j\sqrt{Z^{'}Y^{'}} \right )L=NZ \] 因此对于周期性平行板传输线而言，其所对应的终端短路和开路时的输入阻抗可表示为： \[ \left\{\begin{matrix} Z_{in}^{open}=\frac{1}{j\omega NC_{R}}\\ Z_{in}^{short}=\frac{N}{j\omega C_{L}}+j\omega NL_{R} \end{matrix}\right. \] 为使得周期性平行板传输线工作在谐振频率\(f_{MZR}\)，传输线终端必须短路。此时\(β_{a}\)等于0，平行板传输线上电流相位处处相同。

Huygen’s Metasurface Antenna

香港城市大学目前正在研究的是一种被称为Huygen’s Metasurface Antenna的平面超材料天线，通过周期性排布的天线单元来改变入射电磁场的反射方向，这种天线的基本原理是通过周期性结构改变天线的进场电磁波分布，从而实现特殊的电磁传播。目前，通过仿真和实验验证分析了这种天线结构可以实现电磁波信号100%的沿入射方向进行反射。同时，通过调整周期性天线单元的大小以及相互之间的距离，这种天线还可以实现任意角度的电磁反射，相当于在电磁波入射点将电磁能量全部吸收，然后在传输线上传输一定地距离，然后在全部发射。这种工作模式下，天线的工作效率同样可以达到100%。 在Huygen’s Metasurface Antenna的基础上，香港城市大学的教授还研制了Huygen’s box，这是一种立体结构的盒子，同样是利用周期性的馈电单元来实现盒子内得到任意分布的电磁场结构，而不受场外电磁场分布的影响。

介质谐振器天线

介质谐振器天线（Dielectric Resonator Antenna）是一种谐振式天线，由低损耗、高介电常数的微波介质材料构成，其谐振频率由谐振器尺寸、形状和相对介电常数所决定。介质谐振器天线可用多种结构简单的馈电方式激励，如探针、微带线等。与传统的金属天线不同，介质谐振器天线是利用电磁场在介质材料边缘的场分布形成辐射的。 香港城市大学目前基于水、GaAs、VO2等材料设计了一系列介质谐振器天线，可以通过天线之间的相互作用来提高成像质量、通过不同层之间介质介电常数的不同来控制Ex与Ey的相位差从而实现圆极化、通过新材料在不同电压情况下表现的短路/开路特性来实现天线工作状态的自动调节、通过多端口馈电形式使得天线在不同端口馈电情况下工作状态不同等，并取得一系列非常好的成果。
